# aarch64_u16all
Ubuntu 16.04 aarch64 image with all services

Services installed on the image
- couchdb 1.6.0
- elasticsearch 6.2.4
- memcached 1.5.12
- mongo 4.0.3
- neo4j 3.3.9
- postgres 9.5
- rabbitmq 3.6.15
- redis 5.0.0 (chris-lea ppa)
- selenium 3.141.5
- sqlite 3.11.0


---

Services **not** supported on this image 
- mysql
- cassandra
- rethinkdb
- riak
