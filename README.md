# aarch64_u16all
Ubuntu 16.04 aarch64 image with all services

Services installed on the image
- couchdb 1.6.0
- elasticsearch 6.1.1
- memcached 1.4.25
- mongo 3.6.2
- neo4j 3.3.1
- postgres 9.5
- rabbitmq 3.6.14
- redis 4.0.8 (chris-lea ppa)
- selenium 3.8.1
- sqlite 3.11.0


---

Services **not** supported on this image 
- mysql
- cassandra
- rethinkdb
- riak
